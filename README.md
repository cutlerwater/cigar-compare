# CigarCompare

Luxury dark-themed cigar price comparison site for US retailers.

CigarCompare allows users to compare cigar prices across multiple US retailers,
including Singles, 5-Packs, and Boxes — ranked by true value per stick.

---

## 🚬 Overview

CigarCompare is a frontend prototype for a Trivago-style cigar comparison platform.

Features:

- Premium dark luxury UI
- Box / 5-Pack / Single tab switching
- "In stock only" filtering
- Sorting by:
  - Best value (price per stick)
  - Lowest total price
  - Most recent update
- Dynamic rendering from sample JSON (no backend required)
- Contact, Privacy, and Terms pages
- 21+ disclaimer ready
- Designed for future backend integration
